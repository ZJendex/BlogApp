# BlogApp
An blog app build on RESTful API taught by Colt Steele
* Node JS v14.3.0
* Express v4.17.1
* MongoDB 
## Features
1. RESTful API
2. CRUD Operation
3. Sanitized content
#### Dependencies
>mongoose, express, ejs, express-sanitizerm, body-parser, method-override
